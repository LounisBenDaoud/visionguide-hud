# VisionGuide AI - Backend 

This is the FastAPI server that powers the "VisionGuide AI" prototype. It uses Gemini 2.5 Flash to analyze industrial server racks and provides real-time thermal telemetry.

##  Quick Start 
### 1. Setup Environment
```bash
# Clone the repo and enter the folder
cd visionguide-backend

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # Mac/Linux
.\venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
